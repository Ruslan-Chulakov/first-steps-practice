Введение в CSS

CSS (Cascading Style Sheets, каскадные таблицы стилей)
- это язык для описания и изменения внешнего вида элементов. HTML используется для определения структуры и семантики содержимого документа, а CSS для его оформления и позиционирования.

Синтаксис

Блок CSS-кода называется правилом, которое состоит из селектора и группы объявлений в фигурных скобках.

Селектор указывает браузеру на то, к каким элементам нужно применить стили из этого правила. Каждое объявление состоит из пары свойство: значение;, где свойство и его значение разделяются двоеточием с пробелом и обязательно заканчиваются точкой с запятой. Они описывают что именно нужно сделать со стилями элемента.

Подключение стилей

Есть три способа добавить стили к HTML-документу: встроенные стили (inline styles), встроенная таблица стилей (embedded stylesheet) и внешняя таблица стилей (external stylesheet). У каждого способа есть свои преимущества, недостатки, ограничения и сфера применения.

Встроенные стили

Встроенные (инлайн) стили задаются тегу прямо в атрибуте style. Их невозможно масштабировать, сложно переопределять и использовать повторно, поэтому они применяются в очень редких случаях. Например, для динамических стилей которые устанавливаются во время выполнения JavaScript кода, когда мы заранее не знаем какое будет значение свойства.

Встроенная таблица стилей

При таком подходе CSS-правила добавляются внутрь тега <style> в шапке документа. Их невозможно использовать повторно на других страницах и сложно масштабировать и поддерживать.

ИНТЕРЕСНО

Иногда этот подход используется для оптимизации скорости отрисовки страницы, добавляя самые важные стили прямо в HTML-документ. Эта продвинутая техника называется Critical CSS.

Внешняя таблица стилей

Внешний CSS-код легко масштабировать, поддерживать и использовать повторно на других страницах, поэтому это стандарт подключения стилей. В проекте создаётся отдельный файл стилей с расширением .css и подключается в HTML-документе.

  • На одном уровне с index.html создаётся папка css, а внутри неё файл стилей styles.css.
  • В теге <head>, используя тег <link>, созданный ранее файл подключается.
  • В атрибуте href указывается путь к файлу стилей относительно HTML-документа.
  • В атрибуте rel указывается тип подключаемого документа - stylesheet (таблица стилей).

ПОЛЕЗНО

Можно подключить сколько угодно CSS-файлов, добавляя для каждого тег <link>.
